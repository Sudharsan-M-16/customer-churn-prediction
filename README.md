Built an end-to-end churn prediction system using leakage-safe preprocessing, stratified train/validation/test splits, probability-based decision making, and business-profit–optimized threshold selection. Compared linear and nonlinear models under identical pipelines, showed that logistic regression best matched the data’s structure, and further improved performance via probability calibration. Final model selection was based on unseen profit rather than accuracy, producing a deployable, interpretable, and profit-maximizing system.
